# LKC-systemet - Brukerdokumentasjon

Velkommen til brukerdokumentasjonen for Larvik Kommune Catering (LKC) sitt cateringsystem.

## Om systemet

LKC-systemet er en komplett løsning for catering-virksomhet som inkluderer:
- Oppskriftsadministrasjon med automatisk beregning av næringsverdier
- Menyplanlegging og ukemenyer
- Kunde- og ordrehåndtering
- Produktkatalog med Matinfo-integrasjon
- Etikettdesign og utskrift (inkludert Zebra-skrivere)
- Rapporter og statistikk
- Multi-bruker støtte med roller og tilgangskontroll

## Kom i gang

### Innlogging

**Første gang du logger inn:**
1. Gå til systemets nettside
2. Velg innloggingsmetode:
   - **Google-innlogging**: Klikk "Logg inn med Google" og velg Google-kontoen din
   - **E-post/passord**: Skriv inn e-post og passord (hvis du har fått dette av administrator)
3. Første gang må en administrator godkjenne kontoen din
4. Du får en e-post når kontoen er godkjent

**Viktig**: Hvis du ikke får tilgang innen 24 timer, kontakt din systemadministrator.

### Navigasjon i systemet

Hovedmenyen ligger på venstre side av skjermen og inneholder:

| Menyvalg | Beskrivelse |
|----------|-------------|
| **Hjem** | Dashboard med oversikt og hurtigstatistikk |
| **Kunder** | Kundeadministrasjon og kundeinformasjon |
| **Ordrer** | Ordrehåndtering og ordrehistorikk |
| **Menyer** | Ukemenyer og menyplanlegging |
| **Oppskrifter** | Oppskriftsbibliotek og oppskriftsadministrasjon |
| **Produkter** | Produktkatalog og Matinfo-søk |
| **Etiketter** | Etikettdesign og utskrift |
| **Rapporter** | Statistikk og rapporter |
| **Innstillinger** | Systeminnstillinger og konfigurasjon |

### Brukergrensesnitt

- **Tema**: Bytt mellom lyst og mørkt tema ved å klikke sol/måne-ikonet i sidemenyen
- **Søk**: De fleste sider har søkefelt øverst for rask tilgang
- **Filter**: Bruk filterfunksjoner for å finne spesifikk informasjon
- **Hurtigtaster**:
  - `Escape`: Lukk dialogbokser/modaler
  - `Enter`: Bekreft handlinger
  - `Ctrl/Cmd + S`: Lagre (der det er relevant)
