# Rapporter og Innstillinger

Lær hvordan du genererer rapporter og konfigurerer systemet.

## Del 1: Rapporter

Rapportmodulen gir deg innsikt i drift, salg, produksjon og økonomi.

### Oversikt over tilgjengelige rapporter

| Kategori | Rapporter | Beskrivelse |
|----------|-----------|-------------|
| **Salg** | Omsetningsrapport, Produktsalg, Kundeanalyse | Salgsdata og trender |
| **Produksjon** | Produksjonsvolum, Ingrediensforbruk, Effektivitet | Produksjonsdata |
| **Økonomi** | Lønnsomhetsanalyse, Kostnadsoversikt, Budsjett | Økonomiske nøkkeltall |
| **Kunder** | Kundestatistikk, Ordrefrekvens, Kundetilfredshet | Kundeinnsikt |
| **Meny** | Populære retter, Ernæringsanalyse, Allergenoversikt | Menydata |
| **Lager** | Lagerstatus, Svinn, Innkjøpsbehov | Lagerdata (hvis aktivert) |

---

## Generere rapport

### Steg-for-steg

1. **Naviger til Rapporter**
   - Klikk på **Rapporter** i venstre meny

2. **Velg rapporttype**
   - Klikk på ønsket rapportkategori
   - Velg spesifikk rapport

3. **Sett parametere**

   Vanlige parametere:

   | Parameter | Beskrivelse | Eksempel |
   |-----------|-------------|----------|
   | **Periode** | Fra-dato til-dato | 01.01.2026 - 31.01.2026 |
   | **Sammenligningsperiode** | Sammenlign med tidligere periode | Forrige måned, Samme måned i fjor |
   | **Kunde(r)** | Spesifikke kunder eller alle | Alle, eller velg spesifikke |
   | **Produkt(er)** | Spesifikke produkter | Alle, kategori, eller spesifikke |
   | **Gruppe etter** | Hvordan gruppere data | Dag, uke, måned, kunde, produkt |
   | **Sortering** | Sortere etter | Verdi, navn, dato |

4. **Forhåndsvisning**
   - Klikk **"Forhåndsvis"**
   - Se rapporten på skjermen
   - Juster parametere om nødvendig

5. **Eksporter**
   - Klikk **"Eksporter"**
   - Velg format:
     - **PDF**: For utskrift og arkivering
     - **Excel**: For videre analyse
     - **CSV**: For import til andre systemer
   - Klikk **"Last ned"**

## Salgsrapporter

### Omsetningsrapport

Viser total omsetning over valgt periode.

**Inkluderer**:
- Total omsetning (ekskl. og inkl. mva)
- Omsetning per dag/uke/måned
- Vekst sammenlignet med forrige periode
- Graf over utvikling
- Topp 10 produkter etter omsetning
- Omsetning per kundegruppe

**Bruksområde**:
- Månedlig rapportering til ledelsen
- Budsjetoppfølging
- Identifisere trender

### Produktsalg

Detaljert oversikt over hva som selges.

**Inkluderer**:
- Antall solgte enheter per produkt
- Omsetning per produkt
- Gjennomsnittspris
- Populære og upopulære produkter
- Sesongvariasjoner

**Bruksområde**:
- Produktporteføljeanalyse
- Prissetting
- Menyplanlegging

### Kundeanalyse

Innsikt i kundebase og kjøpsadferd.

**Inkluderer**:
- Antall aktive kunder
- Nye kunder i perioden
- Omsetning per kunde
- Gjennomsnittlig ordrestørrelse
- Ordrefrekvens
- Kundelojalitet

**Bruksområde**:
- Kundeoppfølging
- Identifisere viktige kunder
- Kundebevaring

## Produksjonsrapporter

### Produksjonsvolum

Oversikt over produsert volum.

**Inkluderer**:
- Antall porsjoner produsert
- Produksjon per rett/produkt
- Produksjon per dag/uke
- Kapasitetsutnyttelse
- Produksjonstid (hvis registrert)

**Bruksområde**:
- Kapasitetsplanlegging
- Bemanningsplanlegging
- Effektivitetsanalyse

### Ingrediensforbruk

Detaljert oversikt over ingrediensbruk.

**Inkluderer**:
- Forbruk per ingrediens
- Kostnad per ingrediens
- Sammenligning med budsjett
- Svinn (hvis registrert)
- Forbrukstrend

**Bruksområde**:
- Innkjøpsplanlegging
- Kostnadskontroll
- Svinnreduksjon

### Effektivitetsrapport

Analyse av produksjonseffektivitet.

**Inkluderer**:
- Produksjon per ansatt-time
- Kostnad per porsjon
- Svinnprosent
- On-time delivery (leveringspresisjon)
- Kvalitetsindikatorer

**Bruksområde**:
- Kontinuerlig forbedring
- Benchmarking
- Prestasjonsmåling

## Økonomiske rapporter

### Lønnsomhetsanalyse

Viser lønnsomhet per produkt, kunde, og periode.

**Inkluderer**:
- Salgsinntekter
- Varekostnad (COGS)
- Dekningsbidrag
- Dekningsgrad (%)
- Lønnsomhet per produkt/kunde

**Bruksområde**:
- Identifisere lønnsomme produkter
- Prissetting
- Produktportefølje-optimalisering

### Kostnadsoversikt

Detaljert kostnadsoversikt.

**Inkluderer**:
- Varekostnad
- Ingredienskostnader
- Emballasjekostnader
- Andre direkte kostnader
- Kostnadsutvikling over tid

**Bruksområde**:
- Kostnadskontroll
- Budsjettering
- Prisforhandlinger med leverandører

### Budsjettrapport

Sammenligning mot budsjett.

**Inkluderer**:
- Budsjett vs. faktisk
- Avvik i kroner og prosent
- Prognose for resten av perioden
- Graf over utvikling
- Kommentarfelt for avviksforklaring

**Bruksområde**:
- Månedlig økonomistyring
- Revidere prognoser
- Rapportering til ledelsen

## Kunderapporter

### Kundestatistikk

Overordnet kundestatistikk.

**Inkluderer**:
- Totalt antall kunder
- Aktive vs. inaktive
- Nye kunder i perioden
- Mistet kunder
- Kundevekst (%)

### Ordrefrekvens

Analyse av hvor ofte kunder bestiller.

**Inkluderer**:
- Gjennomsnittlig tid mellom ordrer
- Kunder etter ordrefrekvens (daglig, ukentlig, etc.)
- Kunder med lav aktivitet (risiko for churn)

**Bruksområde**:
- Identifisere kunder å følge opp
- Kundeoppbevaring

## Menyrapporter

### Populære retter

Oversikt over mest og minst populære retter.

**Inkluderer**:
- Topp 20 mest solgte retter
- Bunnen 20 minst solgte
- Solgt volum og omsetning
- Sesongvariasjoner
- Kundeevalueringer (hvis registrert)

**Bruksområde**:
- Menyoptimalisering
- Identifisere retter som kan fjernes
- Markedsføring av populære retter

### Ernæringsanalyse

Analyse av ernæringsmessig verdi av menyer.

**Inkluderer**:
- Gjennomsnittlig energi per måltid
- Fordeling av protein, karbohydrater, fett
- Sammenligning mot anbefalinger
- Menybalanse-score
- Mangel/overskudd av næringsstoffer

**Bruksområde**:
- Sikre ernæringsmessig balanse
- Dokumentasjon for kunder
- Ernæringsfaglig kvalitetssikring

### Allergenoversikt

Oversikt over allergener i menyer og produkter.

**Inkluderer**:
- Andel retter med spesifikke allergener
- Tilgjengelighet av allergen-frie alternativer
- Allergenvarsel-statistikk
- Kunder med allergenrestriksjoner

**Bruksområde**:
- Sikre allergenhåndtering
- Planlegge allergen-frie alternativer
- Dokumentasjon for myndigheter

## Automatiske rapporter

Sett opp rapporter som sendes automatisk på e-post.

### Opprette automatisk rapport

1. Gå til **Rapporter** > **"Automatiske rapporter"**
2. Klikk **"Ny automatisk rapport"**
3. Fyll inn:

   | Felt | Beskrivelse |
   |------|-------------|
   | **Rapportnavn** | Navn på rapport |
   | **Rapporttype** | Velg type rapport |
   | **Frekvens** | Daglig, ukentlig, månedlig |
   | **Dag og tid** | Når skal den sendes? |
   | **Parametere** | Sett standardverdier |
   | **Format** | PDF, Excel, eller begge |
   | **Mottakere** | E-postadresser (kommaseparert) |

4. Klikk **"Lagre"**

Rapporten sendes nå automatisk til valgte mottakere.

### Administrere automatiske rapporter

1. Gå til **Rapporter** > **"Automatiske rapporter"**
2. Se liste over aktive rapporter
3. Handlinger:
   - **Rediger**: Endre innstillinger
   - **Pause**: Stopp midlertidig
   - **Slett**: Fjern permanent
   - **Send nå**: Send umiddelbart

## Profilinnstillinger

Gå til **Profil** (klikk på ditt navn øverst til høyre).

### Personlig informasjon

- **Navn**: Fullt navn
- **E-post**: E-postadresse (brukes til pålogging)
- **Telefon**: Telefonnummer
- **Profilbilde**: Last opp profilbilde

### Preferanser

- **Språk**: Velg norsk, engelsk, etc.
- **Tema**: Lyst eller mørkt tema
- **Tidsone**: Velg tidsone
- **Datoformat**: DD.MM.YYYY eller MM/DD/YYYY
- **Valuta**: NOK, EUR, etc.

### Varslinger

Velg hvilke varsler du vil motta:
- ☑️ Nye ordrer
- ☑️ Ordreendringer
- ☑️ Lagervarsel
- ☑️ Rapporter klare
- ☑️ Systemoppdateringer

Velg kanal:
- E-post
- Push-varsler (mobilapp)
- SMS (hvis aktivert)

### Sikkerhet

- **Endre passord**: Bytt passord
- **To-faktor autentisering**: Aktiver 2FA for ekstra sikkerhet
- **Aktive økter**: Se hvor du er logget inn
- **Logg ut alle enheter**: Logg ut fra alle enheter

## Bedriftsinnstillinger

Gå til **Innstillinger** > **Bedrift** (krever admin-tilgang).

### Bedriftsinformasjon

- **Bedriftsnavn**: Offisielt navn
- **Organisasjonsnummer**: Org.nr
- **Adresse**: Forretningsadresse
- **Kontakt**: Telefon og e-post
- **Logo**: Last opp bedriftslogo (brukes på rapporter og etiketter)

### Mva-innstillinger

- **Standard mva-sats**: 25%, 15%, 0%
- **Mva-nummer**: Mva-registreringsnummer

### Betalingsinnstillinger

- **Betalingsbetingelser**: Standard (f.eks. 30 dager)
- **Fakturapreferanser**: Mal, footer, vilkår

## Brukeradministrasjon

Gå til **Innstillinger** > **Brukere** (krever admin-tilgang).

### Legge til ny bruker

1. Klikk **"Ny bruker"**
2. Fyll inn:
   - **Navn**: Brukerens navn
   - **E-post**: E-postadresse (brukes til pålogging)
   - **Rolle**: Velg rolle (se under)
   - **Avdeling**: Velg avdeling (valgfritt)
3. Klikk **"Send invitasjon"**
4. Brukeren får e-post med invitasjon til å aktivere konto

### Brukerroller

| Rolle | Tilganger |
|-------|-----------|
| **Administrator** | Full tilgang til alt |
| **Leder** | De fleste funksjoner, men ikke systeminnstillinger |
| **Kjøkken** | Oppskrifter, menyer, produksjon |
| **Salg** | Kunder, ordrer, rapporter |
| **Lager** | Produkter, lager, innkjøp |
| **Betraktermodus** | Kun lesetilgang |

### Administrere brukere

- **Deaktiver bruker**: Fjern tilgang uten å slette
- **Endre rolle**: Oppgrader/nedgrader tilganger
- **Tilbakestill passord**: Send passordtilbakestilling
- **Slett bruker**: Permanent sletting

## Integrasjoner

Gå til **Innstillinger** > **Integrasjoner**.

### Matinfo.no

Integrasjon med Matinfo-databasen.

**Konfigurasjon**:
1. Klikk **"Konfigurer Matinfo"**
2. API-nøkkel er forhåndskonfigurert
3. Velg synkroniseringsinnstillinger:
   - **Automatisk synkronisering**: Av, daglig, ukentlig, månedlig
   - **Dag og tid**: Når skal synkronisering kjøre?
   - **Overstyr manuelle endringer**: Ja/nei
4. Klikk **"Lagre"**

### Økonomisystem

Koble til regnskapssystem (f.eks. Tripletex, Visma, Fiken).

**Konfigurasjon**:
1. Velg økonomisystem
2. Klikk **"Koble til"**
3. Godkjenn tilgang i økonomisystemet
4. Velg hva som skal synkroniseres:
   - Kunder
   - Produkter
   - Fakturaer
   - Betalinger
5. Klikk **"Lagre"**

### E-post

Konfigurer e-postintegrasjon.

**SMTP-innstillinger**:
- SMTP-server
- Port
- Brukernavn
- Passord
- Kryptering (TLS/SSL)

### Andre integrasjoner

- **Betalingsgateway**: Stripe, Vipps, etc.
- **SMS-gateway**: For SMS-varsler
- **Analyse**: Google Analytics, Matomo
- **Kalender**: Google Calendar, Outlook

## Sikkerhetsinnstillinger

Gå til **Innstillinger** > **Sikkerhet** (krever admin-tilgang).

### Passordregler

Sett passordkrav:
- **Minimum lengde**: 8-16 tegn
- **Krav til kompleksitet**:
  - ☑️ Store bokstaver
  - ☑️ Små bokstaver
  - ☑️ Tall
  - ☑️ Spesialtegn
- **Passord utløper etter**: 90, 180, 365 dager, eller aldri

### To-faktor autentisering (2FA)

- **Påkrev 2FA**: For alle brukere, eller valgfritt
- **2FA-metode**: App (Google Authenticator), SMS, e-post

### Øktinnstillinger

- **Økttimer**: Automatisk utlogging etter inaktivitet (15, 30, 60 min)
- **Husk meg-varighet**: Hvor lenge "Husk meg" varer (7, 30, 90 dager)

### IP-whitelist

Begrens tilgang til spesifikke IP-adresser (valgfritt):
1. Klikk **"Legg til IP-adresse"**
2. Legg til tillatte IP-adresser eller IP-ranges
3. Aktiver IP-whitelist

## Systeminnstillinger

Gå til **Innstillinger** > **System** (krever admin-tilgang).

### Generelt

- **Systemnavn**: Eget navn på systemet
- **Tidsone**: Standard tidsone
- **Språk**: Standard systemspråk

### Database

- **Backup**: Automatisk backup (daglig, ukentlig)
- **Backup-tid**: Når skal backup kjøre?
- **Oppbevaringstid**: Hvor lenge skal backups beholdes?

### API

- **API-tilgang**: Aktiver/deaktiver API
- **API-nøkler**: Administrer API-nøkler for integrasjoner
- **Rate limiting**: Begrens antall API-kall

### Logging

- **Loggingsnivå**: Debug, Info, Warning, Error
- **Loggoppbevaring**: Hvor lenge skal logger beholdes?
- **Eksporter logger**: Last ned logger for feilsøking

## Best practices

### Sikkerhet
- Aktiver 2FA for alle brukere
- Bruk sterke passord
- Gjennomgå brukertilaverage and delete inactive accounts
- Backup data regelmessig

### Varsler
- Sett opp relevante varsler
- Ikke få for mange varsler (varselutmattelse)
- Test varsler etter oppsett

### Rapporter
- Lag automatiske rapporter for faste behov
- Eksporter viktige rapporter for arkivering
- Del rapporter med relevante personer
